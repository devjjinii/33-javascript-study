## this, call, apply, bind

```js
function fnc() {
    return 'hello';
}

fnc();
fnc.call();
fnc.apply();
```

* call : (실행)

* apply : 배열로 사용 (실행)
    * ...연산자
```js
const array = ['a', 'b'];
const elements = [1, 2, 3];

array.push.apply(array, elements);
console.log(array); // ["a","b",1,2,3]
```

* bind : this는 내가 정한 object로 고정 (연결)
